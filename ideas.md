# Westen Intel — Design Brainstorm

## Chosen Approach: Classified Dossier / Cold War Noir

<response>
<text>
**Design Movement:** Cold War Noir / Classified Intelligence File

**Core Principles:**
1. Everything looks like it was pulled from a redacted CIA dossier — stamped, filed, and barely declassified
2. Text is king: voiceovers are the product, so typography does the heavy lifting
3. Dark Miami heat — deep charcoal backgrounds with amber/burnt-orange accents evoking both Miami sun and danger
4. Functional brutalism in the layout — asymmetric, grid-breaking, like a corkboard of evidence

**Color Philosophy:**
- Background: near-black charcoal `#0d0e0f` — the inside of a classified folder
- Primary accent: burnt amber `#c8600a` — Westen's jacket, Miami sunsets, danger
- Secondary: faded olive `#4a5240` — military, field ops
- Text: off-white `#e8e0d0` — aged paper, typewriter output
- Highlight: blood red `#8b1a1a` — CLASSIFIED stamps, alerts

**Layout Paradigm:**
- Hero: full-bleed dark background with a single typewriter-animated opening voiceover
- Theme sections: staggered card grid with "DOSSIER" tab headers
- Each card: looks like a torn intelligence report — ragged edges, monospace font, subtle paper texture
- Sidebar filter: looks like a manila folder index

**Signature Elements:**
1. CLASSIFIED / REDACTED stamps on theme headers
2. Typewriter animation on featured quotes
3. Thin amber underlines instead of borders — like pen underlines on a document

**Interaction Philosophy:**
- Hover reveals "declassified" content — opacity shifts, stamps fade
- Search feels like running a query on a field terminal
- Filtering by season/theme feels like flipping through a dossier

**Animation:**
- Typewriter effect on hero quote (character by character)
- Fade-in with slight upward drift for cards
- Stamp animation when hovering theme headers

**Typography System:**
- Display: `Special Elite` (Google Fonts) — typewriter/classified document feel
- Body: `IBM Plex Mono` — terminal/intelligence report
- Accent labels: `Oswald` condensed — military stencil feel
